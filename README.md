# Develop
## Getting Started
```
npm install
docker-compose up --force-recreate --always-recreate-deps --abort-on-container-exit --remove-orphans --renew-anon-volumes
```
## Getting Stopped
```
docker-compose down
```
## Build Contracts
```
npm run build-contracts
```
## Build UI
```
npm run build-ui
```
## Watch UI
```
npm run watch-ui
```
## Serve UI
```
npm run serve-ui
```
